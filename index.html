  1. 在本地执行 `git init`、`git add .` 和 `git commit`，生成首个版本。
  2. 到 GitHub 新建同名仓库，把远程地址绑定到 `origin`。
  3. 推送 `main` 分支后，在仓库设置里启用 GitHub Pages 或 Actions 发布。
